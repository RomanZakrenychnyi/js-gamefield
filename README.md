Це простий приклад ігрового поля, яке складається з комірок
У кожній комірці знаходиться ігровий елемент (символ)

При кліку на комірку:
- знаходиться вся пов’язана група таких самих елементів
- якщо група складається з 2 або більше комірок — вона видаляється

Реалізація виконана на чистому HTML + CSS + JavaScript, без фреймворків

# Перший варіант як запустити проєкт:

1. Склонуйте репозиторій або завантажте файли
2. Відкрийте файл 'index.html' у браузері (Chrome, Firefox і т.д.)

Ніякої додаткової збірки або налаштувань не потрібно

## Другий спосіб запуску проєкта - через локальний сервер:

Якщо ви користуєтесь VS Code:

1. Відкрийте папку з проєктом
2. Встановіть розширення Live Server
3. Клікніть правою кнопкою по 'index.html'
4. Оберіть Open with Live Server, або відкривши файл 'index.html' клікність на нижній панелі VSCode справа --> 'Go Live'

Проєкт відкриється у браузері автоматично
